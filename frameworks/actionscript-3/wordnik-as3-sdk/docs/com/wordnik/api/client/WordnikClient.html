Packagecom.wordnik.api.client
Classpublic class WordnikClient
InheritanceWordnikClient flash.events.EventDispatcher

The Wordnik Api client class to be used by the consumers of this SDK to invoke the Wordnik REST API. It supports individual calls and a batch method to bunch multiple calls in one request.

Typical usage looks like this: 
 // Instantiate a WordnikClient
 const wordnikCredentials = new ApiUserCredentials(apiToken, null, apiHostName, userId);
 const wordnikClient = new WordnikClient(wordnikCredentials);
 
 // Add listners to get response back 
 wordnikClient.addEventListener(ApiClientEvent.WORD_LOOK_UP_COMPLETE_EVENT, onWordRequestCompleted);
 wordnikClient.addEventListener(ApiClientEvent.SUGGESTIONS_RESPONSE_EVENT, onWordSuggestion);
 
 // Make calls
 wordnikClient.lookupWord("Dracula");
 wordnikClient.getSpellingSuggestions("Zeebra");
Or to make multiple calls in one shot use the batch feature: 
 // Create a batch
 const batch: Batch = new Batch();
 batch.add(new OpGetDefinitions(2));
 batch.add(new OpGetExamples());
 batch.add(new OpGetRelatedWords());
 
 // Execute it
 wordnikClient.execute("gargantuan", batch);
 wordnikClient.addEventListener(ApiClientEvent.BATCH_EXECUTE_RESPONSE_EVENT, onBatchResponse);
Under the sample folder of as3-sdk distribution, you'll find a sample app which calls all APIs. You'll need to set your yourname, password and api-key in the class Credntials before you can use it.
